# Repositório — Exemplo

Este repositório foi preparado como exemplo e contém alguns arquivos de teste (HTML, JS, Python e TXT).

Arquivos principais:
- `teste.html` — página de teste
- `index.html` — página simples criada automaticamente
- `teste.js` — script de teste
- `teste.py` — script de exemplo

Como usar:
1. Abra `index.html` no navegador para ver a página de resumo.
2. Edite `teste.html` e outros arquivos conforme necessário.
